## API Endpoints

- `GET /api/users`: Get all users.
- `GET /api/users/{id}`: Get a specific user by ID.
- `POST /api/users`: Create a new user.
- `PUT /api/users/{id}`: Update an existing user.
- `DELETE /api/users/{id}`: Delete a user.
